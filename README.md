RoomieSplit is a modern, intuitive web application designed to make sharing expenses with roommates effortless. Built with React, TypeScript, and Tailwind CSS, it helps you track bills, split costs, and settle debts with ease.

## âœ¨ Features

- **Dashboard Overview**: Get a clear snapshot of total expenses, your share, and who owes what at a glance.
- **Expense Tracking**: Easily add expenses with flexible splitting options (equal, percentage, or custom amounts).
- **Roomie Management**: Manage roommates with custom names and color-coding for easy identification. The "Me" user is always present to track your personal standing.
- **Smart Debt Simplification**: The app calculates the simplest way to settle debts, minimizing the number of transactions needed.
- **Settle Up**: A guided flow to clear debts when payments are made.
- **History Log**: View a detailed history of all past expenses and settlements.
- **Responsive Design**: Fully responsive UI that works great on desktop and mobile.
- **Modern UI/UX**: Polished interface with smooth animations powered by Framer Motion.

Follow these steps to get the project up and running locally.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm (or yarn/pnpm)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ddl29/roomieSplit.git
   cd roomieSplit
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
